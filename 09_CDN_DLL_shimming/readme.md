# cdn_dll_shimming

## cdn

### 通过 publicPath

- 通过 这样配置一些 cdn 文件的前缀 然后打包的时候进行一个处理
  - 需要我有自己的 cdn 资源
  - 所有的资源都在 cdn 中

### 只有第三方的库进行 cdn

- externals
  - 注意 这个一定要在 index.html 中引入 cdn 的脚本文件
